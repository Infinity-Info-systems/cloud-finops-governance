# Cloud Cost Governance

## Purpose

Cloud cost governance defines how spending is owned, reviewed, explained, and improved.

## Core Rules

- assign a cost owner for each major service or platform
- define allocation logic for shared and direct spend
- review forecast variance regularly
- require action plans for material overspend
- track savings claims after implementation
- review unallocated spend as a governance issue

## Governance Questions

- Who approves budget changes?
- Who owns shared platform spend?
- How are unallocated costs handled?
- What is the review cadence for major spend variances?

## Evidence To Collect

- budget review notes
- cost allocation rules
- tagged resource samples
- exception approvals
- forecast reports

## Outcome

Cloud cost governance should make spending accountable, explainable, and easy to review with leadership.
