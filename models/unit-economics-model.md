# Unit Economics Model

## Purpose

The unit economics model connects cloud spend to business outcomes so leaders can understand value per transaction, customer, workload, or service.

## Common Measures

- cost per transaction
- cost per active user
- cost per environment
- cost per workload
- cost per business unit

## Interpretation

- use trending data instead of a single month
- compare like for like
- avoid using unit economics without context

## Use

- show business value alongside cost
- support investment decisions
