# Cost Optimization Model

## Purpose

This model groups optimization opportunities into repeatable categories that can be reviewed and tracked.

## Opportunity Types

- rightsizing
- scheduling and shutdown
- commitment optimization
- storage tiering
- architecture optimization
- allocation cleanup

## Prioritization Factors

- savings potential
- implementation effort
- risk to service
- speed to value
- dependency on other changes

## Use

- rank opportunities by value and risk
- track actions to verified savings
- avoid treating theoretical savings as realized savings

## Model Rule

Optimization should be judged on verified results, not just the size of the opportunity list.
